Detailed Description

Class representing an OpenGL framebuffer.

This class is primarily used in the deferred rendering pipeline, where a series of renders to different framebuffers is chained together. Generally, this class can be used at any stage of OpenGL rendering, either to read from or write to.

See also
cannon::graphics::DeferredRenderer
