Detailed Description

Class representing a render pass in the deferred rendering pipeline.

This class represents a stage in the deferred rendering pipeline, encapsulating a framebuffer, shaders to apply, and function that does all of the necessary rendering computation. This class is designed such that later rendering passes can access the results of this rendering pass to do further computations.
