Detailed Description

Class representing a directional light source (e.g., the sun)

This class really just holds the data necessary to render the effect of a directional light in a scene; no geometry is drawn to represent it. It must be applied to a particular piece of geometry in order for the effects of the light to show up in renders.

See also
cannon::graphics::Light, cannon::graphics::PointLight
