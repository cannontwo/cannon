Detailed Description

Class encapsulating an OpenGL Vertex Array Object. Responsible for binding and unbinding the encapsulated OpenGL object. Note that this class cannot be copied because there would be harmful interactions between multiple VertexArrayObjects with the same associated OpenGL object.
