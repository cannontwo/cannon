Detailed Description

Class representing a simple 3D camera able to translate and rotate.

This class implements a simple, fly-style camera. It is adapted from https://learnopengl.com/Getting-started/Camera. The camera is responsible for generating appropriate perspective matrices for OpenGL rendering and is held by graphics classes that provide 3D rendering capabilities.

cannon::graphics::Viewer3D, cannon::graphics::DeferredRenderer
