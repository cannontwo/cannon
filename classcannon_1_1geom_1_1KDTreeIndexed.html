Detailed Description

Class representing a kd-tree which keeps track of the index of inserted points.

This class is an extension on top of a straightforward kd-tree that associates an index with each inserted point. This is useful if the inserted points are also kept in some structure outside of the kd-tree, in which case storing the index for each point obviates the need to search for retrieved nearest neighbor points.
