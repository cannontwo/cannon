Detailed Description

Class representing a point light.

This class embodies the behavior of a point light, which radiates light in all directions. In particular, such a light has ambient, diffuse, and specular colors, as well as a position from which it radiates. The precise behavior of this light is controlled by the shaders used. The position of the point light is represented by a cube.
